# DrRayTracer
A plain ray-traycer.

##accomplished functions:
* basic ray tracing (refracting, reflacting)
* soft shadow
* rectangle lighters
* textures
* anti-alias
* multi-threads
* depth of field
* random diffuse
* obj reader
* KD-tree
